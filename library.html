Kütüphane

Düşüncelerimi besleyen, mesleki gelişimime yön veren yazılı kaynakları derlediğim alan.
Kitaplar, Notion tablomdan GitHub Actions sayesinde otomatik olarak anlık güncellenmektedir.

Sistem Hakkında

Kütüphane verileri doğrudan bir JSON dosyasından çekilmektedir. Bu dosya, bir GitHub Actions scripti ile Notion veritabanından veri alınarak günlük şekilde dinamik olarak güncellenir.
