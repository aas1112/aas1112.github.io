# 🚀 Ahmet Arda Sezgin - Kişisel Portfolyo Web Sayfası

Bu depo, bir **Elektrik ve Elektronik Mühendisi** olan Ahmet Arda Sezgin'in akademik ve profesyonel yeteneklerini, deneyimlerini, güncel projelerini ve aldığı eğitimleri sergilemek amacıyla modern web teknolojileri (HTML5, CSS3, Vanilla JS) ile geliştirilmiş kişisel portfolyo web sitesini içerir.

![Portfolio Görüntüsü](images/logo.png)

## 📌 Proje Hakkında

Sayfa tasarımı, ziyaretçilere sade ve profesyonel bir deneyim sunmayı amaçlar. Modern ve responsive bir yapıya sahip olan bu web sitesi, hakkımda bölümünden detaylı proje sunumlarına, doğrulanabilen sertifikalara kadar uzanan kapsamlı bir bilgi havuzu barındırır. Web sitesi sayesinde Ahmet Arda Sezgin'in yenilikçi ürünler/sistemler geliştirme sürecindeki yeteneklerini daha yakından ve etkileşimli olarak inceleyebilirsiniz.

Site genel itibariyle aşağıdaki temel sayfalardan oluşmaktadır:
1. **Ana Sayfa (`index.html`):** Kariyer hedefi, eğitim geçmişi (İzmir Ekonomi Üni. Yüksek Lisans/Lisans), deneyimler (Ar-Ge çalışmaları) ve kapsamlı yetenek ağacı.
2. **Projeler (`projects.html`):** Özellikle robotik (ROS/ROS2), makine öğrenmesi ve yapay zeka odaklı (YOLOv5, OpenCV vb.) geliştirilmiş üst düzey otonom sistem projelerinin açıklamaları ve doğrudan kaynak kod linkleri.
3. **Sertifikalar (`certificates.html`):** Yazılım, elektrik/elektronik, görüntü işleme ve yönetim başta olmak üzere dünyanın önde gelen platformlarından edinilen dinamik ve filtrelenebilir sertifikaların sergilendiği arşiv.

## 🛠️ Kullanılan Teknolojiler ve Araçlar
Bu proje dışarıdan ağır bir web framework'ü (React, Next vs.) kullanılmadan tamamen saf web teknolojileri ile performansı ve hızı artırmak üzere tasarlanmıştır.

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid mimarisi)
- **Scripting:** Vanilla JavaScript (ES6+)
- **Dinamik Veri Renderlama:** JSON benzeri objeler ile Javascipt üzerinden dinamik veri entegrasyonu (`certificates.js`)
- **İkonlar ve Fontlar:** Font Awesome, Google Fonts (Inter, Outfit)

## 🎯 Öne Çıkan Özellikler

- **Responsive & Mobil Uyumlu Tasarım:** Bütün cihaz çözünürlükleri (telefon, tablet, PC) ile tamamen uyumlu çalışacak bir dizayn.
- **Kategorilendirilmiş Dinamik Sertifika Havuzu:** JavaScript üzerinden anlık çalıştırılan, kullanıcının filtreler (Yazılım, Yönetim, Görüntü İşleme vb.) üzerinden arama yapabilmesine ve tıklayarak modal ekranda doğrulanmış PDF çıktılarını görmesine olanak veren sistem.
- **Detaylı Proje Kartları:** GitHub repo entegrasyonlu ve doğrudan kullanılan teknolojileri vurgulayan temiz proje kartı stili.
- **Zengin İletişim Footer ve Header:** Hızlı site içi navigasyon, email gönderimi ve sosyal medya hesaplarına geçiş bağlantıları.

## 🚀 Yerel Ortamda Çalıştırma

Eğer projeyi kendi bilgisayarınızda (yerel sunucuda) çalıştırmak isterseniz aşağıdaki adımları takip edin:

1. Repoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/aas1112/aas1112.github.io.git
   ```

2. Proje dizinine girin:
   ```bash
   cd aas1112.github.io
   ```

3. Geliştirme (development) sunucusunu başlatmak için (örn. Python kullanarak):
   ```bash
   python3 -m http.server 8000
   ```
4. Tarayıcınızdan şu adrese gidin: `http://localhost:8000`
